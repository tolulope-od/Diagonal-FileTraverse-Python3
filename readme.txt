Project Title
Traversing Excel Files using OOP(Python3)
-A simple program that reads excel format files, extracts data and stores the data extracted for further operations in Python

Getting Started
You can get a copy of this project by either cloning or downloading the ZIP file to your local machine

Requirements
You will need to install xlrd module for python 3. Run the following code in your command line if you have pip installed

pip install xlrd

Running Tests
This project contains a test Excel format file that displays the result of traversing a file diagonally, storing the cells in a python List element and then displaying the values stored in the List, all using methods defined within a Class.
